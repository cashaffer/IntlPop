IntlPop 3.0 Project Homepage

Welcome to the IntlPop project homepage! This version of IntlPop is written in HTML5/JavaScript.

IntlPop is a simple population simulation, suitable for use in introductory geography courses. IntlPop was originally created in the early 1990's as part of the GeoSim project, written as a C application for the PC. The second version was written in 1999 as a Java applet. This is currently available here.

A public copy of the sourcecode for Intlpop is available through GitHub here.
